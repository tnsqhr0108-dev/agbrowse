Configuration is environment-driven and local.

NamePurpose
BROWSER_AGENT_HOMEHome directory for browser state, sessions, leases, and traces.
CDP_PORTChrome DevTools Protocol port, defaulting to 9222.
AGBROWSE_WEB_AI_AUTO_STARTSet to 0 to fail when Chrome is not running instead of auto-starting.
AGBROWSE_JSON_ERRORSSet to 1 to force JSON error envelopes.
AGBROWSE_PROVIDER_POOL_MAX_PER_KEYOverride warm provider tab pool size per key.

Local-first boundary

agbrowse stores operational state on the local machine. It does not add hosted model routing, provider API-key auth, or cloud browser execution.

Release gatesFirst run
Do not commit local browser homes, traces, downloads, or generated code artifacts.
