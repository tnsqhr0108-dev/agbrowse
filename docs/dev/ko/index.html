개발자 문서

로컬 Chrome을 제어하고 provider Web-AI 워크플로를 실행합니다.

agbrowse는 CDP로 headed Chrome에 붙는 짧은 Node CLI입니다. 브라우저 primitive, ChatGPT/Gemini/Grok Web UI 자동화, code artifact 회수, release evidence를 한 표면에서 다룹니다.

브라우저 런타임

Chrome을 한 번 띄우고 snapshot, click, tab, diagnostics, adaptive fetch를 독립 명령으로 실행합니다.

Web-AI

send, poll, query, watch, session recovery, project sources, source audit를 provider tab 위에서 수행합니다.

코드 artifact

ChatGPT code mode는 strict prompt를 보내고 zip을 headless로 회수한 뒤 PLAN.md를 검증합니다.

릴리즈 근거

structure 문서, release gate, claim audit가 public support label을 코드와 맞춥니다.

먼저 실행

npm install -g agbrowse
agbrowse start --headed
agbrowse snapshot --interactive

기본 코드 모드 호출

agbrowse web-ai code \
  --vendor chatgpt \
  --model thinking \
  --effort standard \
  --prompt "Build a small MVP with tests." \
  --output-zip ./result.zip
