설정은 로컬 환경변수 중심입니다.

이름역할
BROWSER_AGENT_HOMEbrowser state, session, lease, trace 저장 위치.
CDP_PORTChrome DevTools Protocol port. 기본값은 9222.
AGBROWSE_WEB_AI_AUTO_START0이면 Chrome이 없을 때 자동 시작하지 않고 실패합니다.
AGBROWSE_JSON_ERRORS1이면 JSON error envelope를 강제합니다.
AGBROWSE_PROVIDER_POOL_MAX_PER_KEYprovider warm tab pool 크기 override.

Local-first 경계

agbrowse는 동작 상태를 로컬에 저장합니다. hosted model routing, provider API-key auth, cloud browser execution은 추가하지 않습니다.

릴리즈 게이트첫 실행
로컬 browser home, trace, download, generated code artifact는 commit하지 않습니다.
