첫 실행

자동화 전에 로컬 브라우저 계약을 확인합니다.

agbrowse는 숨겨진 브라우저 상태를 가정하지 않습니다. CDP port에 연결하고 현재 target을 읽으며, 필요한 browser/provider 조건이 없으면 fail closed합니다.

체크리스트

  • agbrowse start --headed를 실행합니다.
  • agbrowse status --json으로 CDP port와 headed 상태를 확인합니다.
  • click/type 전에는 agbrowse snapshot --interactive를 실행합니다.
  • Web-AI는 같은 browser profile에서 provider 로그인을 먼저 해 둡니다.
agbrowse status --json
agbrowse tabs --json
agbrowse web-ai status --vendor chatgpt --json

Fail-closed smoke

AGBROWSE_WEB_AI_AUTO_START=0 \
agbrowse web-ai code --vendor gemini --prompt x --json
브라우저 런타임설정
스크립트에서는 JSON output과 명시적 provider flag를 우선 사용합니다.
