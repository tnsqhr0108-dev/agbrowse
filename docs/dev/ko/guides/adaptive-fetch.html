가이드

이미 알고 있는 URL을 제한된 escalation ladder로 읽습니다.

agbrowse fetch는 검색 명령이 아닙니다. 후보 URL이 있을 때 안전 점검과 함께 내용을 추출하는 명령입니다.

agbrowse fetch https://example.com --json
agbrowse fetch https://example.com --selector main --max-bytes 60000
Flag역할
--browser auto|never|requiredisolated browser escalation 제어.
--browser-session none|isolated|existing|user|interactive인증된 사용자 상태 허용 여부 선택.
--identity auto|minimal|chrome요청 header identity 선택.

WAF/auth/paywall signal은 감지하고 보고할 수 있지만 CAPTCHA 우회, 접근 제어 우회, 명시 없는 private credential 사용은 하지 않습니다.
