# MasterBook Integration

This repository is connected to the MasterBook project.

Goal:
수학 기초가 약한 학생을 수능 수학 백분위 85 근처까지 끌어올리기 위한 개인 맞춤형 수능 수학 마스터북 생성 시스템을 구축한다.

Core flow:
조건 읽기 -> 반응 선택 -> 첫 식 세우기 -> 계산 -> 검산 -> 변형 대응

Tools:
- Codex CLI
- GitHub CLI
- AGBROWSE
- AGBROWSE Codex skills: browser, web-ai, vision-click
- Jawcode

Safety:
Do not commit secrets, browser sessions, Codex sessions, GitHub tokens, API keys, or .env files.
